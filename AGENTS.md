# Codex Project Rules

This repository is a related training tool for the AUX AMIS staff manual app.

Cross-project update rules:
- For updates that may also affect the staff manual or other training tools, use the staff manual repo's `docs/project-update-map.md` as the cross-project source of truth.
- This repository's registered role is the table-number quiz. Keep updates limited to table numbers, floor/layout references, and quiz content this tool actually owns.
- User instructions that explicitly include or exclude this repository take priority over the map.
- Do not change other repositories from here unless the user explicitly asks for a cross-project update.
- Preserve the existing file structure and data format in this repository.
- For cross-project work, build/check, commit, and push this repository independently from every other repository.
